## 🏗 Featured Project

### 🏥 NIROG — AI-Powered Healthcare Assistant

> A symptom-based disease prediction system built to bridge the gap between early symptom onset and informed medical guidance — particularly for users without immediate access to a doctor.

**What it does:**
- Accepts user-reported symptoms as structured input
- Runs a trained ML classification model to predict the most probable disease
- Returns a prediction alongside basic health guidance for the detected condition
- Designed for speed and accessibility — no medical background needed to use it
- Tackles a real-world multi-class classification problem: mapping symptom combinations to 40+ disease categories

**The hard problem solved:** Medical symptom data is sparse and noisy — users rarely report the "right" set of symptoms. NIROG handles this through feature encoding that maps free-form symptom selection to model-ready vectors, maintaining prediction reliability even with incomplete inputs.

**Built with:** `Python` · `Scikit-learn` · `Pandas` · `NumPy` · `Streamlit / Flask`
