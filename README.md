# 🏥 NIROG – AI Healthcare Assistant

## 🚨 Problem Statement

Access to quick and reliable healthcare guidance is limited, especially for early disease detection.
People often ignore symptoms or lack immediate medical assistance.

---

## 💡 Solution

NIROG is an AI-powered healthcare assistant that helps users analyze symptoms and get possible disease predictions along with basic health guidance.

---

## ⚙️ Tech Stack

* Python
* Machine Learning
* Pandas / NumPy
* Scikit-learn
* (Add Flask / Streamlit if used)

---

## 🔍 Key Features

* ✅ Symptom-based disease prediction
* ✅ User-friendly interface
* ✅ Fast and accurate results
* ✅ Helps in early detection

---

## ▶️ How to Run

1. Install dependencies
2. Run the main file
3. Enter symptoms
4. Get prediction

---

## 📊 Output

* Predicts possible disease
* Provides basic health insights
